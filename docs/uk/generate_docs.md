# Генерація документації

У проєкті SmartSchool для автоматичної генерації документації використовується **Javadoc**.

## Передумови

Для генерації документації необхідно:
- мати встановлений JDK;
- мати встановлений Maven;
- додати Javadoc-коментарі до публічних класів і методів.

## Команда генерації

Для генерації документації потрібно виконати команду:

```bash
mvn javadoc:javadoc
```

## Результат

Після виконання команди документація буде згенерована у директорії:
target/site/apidocs

Головна сторінка документації знаходиться у файлі:
target/site/apidocs/index.html
