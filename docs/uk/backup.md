# Резервне копіювання

## 1. Поточний стан

* Дані не зберігаються
* Втрачаються після перезапуску

## 2. Майбутня стратегія

При використанні БД необхідно створювати backup.

## 3. Backup БД

pg_dump smartschool > backup.sql

## 4. Відновлення

psql smartschool < backup.sql

## 5. Частота

* щоденно
* перед оновленням

## 6. Висновок

У поточній версії backup не потрібен.
