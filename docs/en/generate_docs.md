# Documentation Generation

In the SmartSchool project, **Javadoc** is used for automatic documentation generation.

## Prerequisites

To generate documentation, the following requirements must be met:
- JDK must be installed;
- Maven must be installed;
- Javadoc comments must be added to public classes and methods.

## Generation Command

To generate documentation, run the following command:

```bash
mvn javadoc:javadoc
